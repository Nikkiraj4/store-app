# 🛒 Store App

A modern, responsive e-commerce storefront built with **React.js** and **Tailwind CSS**. This project focuses on clean UI/UX, real-time product filtering, and seamless navigation.

🚀 **Live Demo:** [https://store-app-techno5.vercel.app/](https://store-app-techno5.vercel.app/)

---

## ✨ Features

* 🔍 **Real-time Search:** Instantly filter through product catalogs.
* 📱 **Fully Responsive:** Optimized for Mobile, Tablet, and Desktop using Tailwind CSS.
* ⚡ **Performance:** Fast loading states with custom `LoadingSpinner` components.
* 🎨 **Modern UI:** Clean aesthetic with a focus on product visibility.

## 🛠️ Tech Stack

* **Frontend:** React.js, JavaScript (ES6+)
* **Styling:** Tailwind CSS, PostCSS
* **Deployment:** Vercel
* **Icons/Assets:** Lucide React / FontAwesome (if applicable)
