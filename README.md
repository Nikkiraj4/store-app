# 🛒 Store App

A responsive, component-based React application that consumes data from the **Fake Store API**. Features search functionality, loading states, error handling, and fully responsive design.

## ✨ Features
- ✅ Fetches products from Fake Store API
- ✅ Real-time search by product name & category
- ✅ Responsive grid layout (mobile, tablet, desktop)
- ✅ Loading & error states
- ✅ Component-based architecture
- ✅ Tailwind CSS for styling
- ✅ Modern UI/UX with animations

## 🛠️ Tech Stack
- **React 18** - Component-based UI library
- **Tailwind CSS** - Utility-first CSS framework
- **Fake Store API** - Public e-commerce API
- **Vite** - Fast development server
